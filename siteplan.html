Individual Project Site Plan

Explore Naija

Site Name

Explore Naija

The name Explore Naija was selected because it clearly represents a platform dedicated to discovering Nigeria’s cultural places, landmarks, and experiences. “Naija” is a widely recognized and informal reference to Nigeria, making the name memorable, friendly, and culturally relevant.

Optional domain availability: explore-naija.com

Site Purpose

The purpose of this website is to provide users with curated information about notable cultural and historical places in Nigeria. The site will dynamically present location details, descriptions, and categories while allowing users to interact with the content, view additional details in modal dialogs, and save preferences using local storage.

Scenarios

  • A visitor wants to know which cultural or historical attractions are available in a specific Nigerian state.
  • A user wants to view more detailed information about a place and save it as a favorite for future reference.

Color Scheme

The following colors were selected to reflect Nigerian culture, nature, and warmth:

  • Primary Color (#0b6e4f – Deep Green): Used for headings, navigation, and major branding elements.
  • Secondary Color (#f4a261 – Warm Orange): Used for accents, borders, buttons, and highlights.
  • Background Color (#f9f9f9 – Light Neutral): Used as the main page background for readability and contrast.

Typography

  • Poppins (Sans-serif): Used for headings and titles to create a modern and bold visual hierarchy.
  • Open Sans (Sans-serif): Used for body text and paragraphs to ensure readability and accessibility.
